A backport of the modern JEI item utility system to Minecraft 1.7.10 for learning.

## Required Dependencies:

- [OKCore (>= 26.07.13.0)](https://github.com/Shigure-Ruiseki/OKCore)

## Features

- Recipe and usage
- Fast item search with filtering
- Support for crafting, furnace.
- Lightweight for 1.7.10
- Designed for compatibility with 1.7.10 modpacks
